# Access Igbo Examples

```js {1}
[ Base URL: igboapi.com/api/v1/ ]
```

***

###### Examples Route

```js copy
GET /examples
```
Get examples in dictionary

 **Parameters**

| Name   | Description |
| :----- | :----- |
| **keyword** *required* | Keyword for querying words (string) |
| **page** | Page for results (integer) |
| **range** | Page for result using [x,y] syntax, max range includes 25 documents (string) |

 **Responses**

 The response for the route will be a plain JSON object similar to JSON below.

 ###### Single example Route


```js copy
GET /examples/{exampleId}
```
Returns a single Example object from the database

 **Parameters**

| Name   | Description |
| :----- | :----- |
| **exampleId** *required* | The example Id (string) |

 **Responses**

 The response for the route will be a plain JSON object similar to this:

 ```
{
  "nsibidi": "",
  "igbo": "Agụ nà-àtakwo mangō ācharọ acha",
  "english": "Agụ is eating unripe mango",
  "associatedWords": [
  ],
  "pronunciation": "https://igbo-api.s3.us-east-2.amazonaws.com/audio-pronunciations/620c4c01ebaecf171e074c89.webm",

  "style": "",

    null
  ],
}
 ```
